# tabasamu-decorators

A modern web application for Tabasamu Decorators - your premier event decoration and planning service.

## Technologies Used

- [Vite](https://vitejs.dev/guide/) - Next Generation Frontend Tooling
- [React](https://react.dev) - JavaScript library for building user interfaces
- [TypeScript](https://www.typescriptlang.org) - Typed JavaScript at Any Scale
- [Tailwind CSS](https://tailwindcss.com) - Utility-first CSS framework
- [HeroUI](https://heroui.com) - Beautiful React UI components
- [Framer Motion](https://www.framer.com/motion) - Production-ready animation library

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm, yarn, pnpm, or bun

### Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/tabasamu-decorators.git
cd tabasamu-decorators
```

Install dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

Create a production build:

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
